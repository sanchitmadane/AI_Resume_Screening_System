# AI Resume Screening & Skill Extraction System

This project demonstrates how Artificial Intelligence can automate resume screening by comparing resumes with a job description using Natural Language Processing.

## 🚀 Features
- Resume similarity scoring using NLP
- TF-IDF Vectorization
- Cosine Similarity Matching
- Candidate Ranking
- Skill Extraction from Job Description
- Visualization of Resume Scores

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- NLP (TF-IDF)

## 📊 Output
The system ranks resumes based on similarity with the job description and recommends the best candidate.
